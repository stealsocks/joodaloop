Useful

Tools and stuff that I recommend.

Software:

Bear: My text editor of choice. A minimal but full-featured markdown editor that does pleasant things like detect links in your clipboard and uses a lightweight tagging system. I would pay for the Pro version but I've I like working offline and have been burnt by sync problems on other writing apps enough that I prefer my editors asynchronous.

Rectangle: A window manager for MacOS.

Hugo: It would be worth it just for the speed, but it's also the simplest, most customizable option I've tried. There's never been a moment where I've wished for a feature and found it lacking. Both 100songs and are built with Hugo. The only reason this site isn't is because I thought there was sentamentality in handcrafting the whole thing. I've heard Pelican is cool too though.

Fish: Command-line autocomplete that just works.
